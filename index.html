HTML

HTML is made up of text content (what you see), markup (what it looks like), references to other documents (eg images or videos) and links to other pages.

Markup

In order to give format to the text in HTML (Markup) we instruct computer using TAGS. Keep in mind that sometimes we refer to computers as "stupid", because they follow instructions exactly as we give them. This gives us a great amount of power, while at the same time we have to be very careful with detail.

BASIC STRUCTURE

< "NAME" > CONTENT < /"NAME" >

OPENING TAG-----------CLOSING TAG

The basic structure formed by opening tag, content and closing tag is called element.

Since there are lots of different tags, it would be very difficult to memorize them all. It is always easier to google them the moment you need it.

A good thing to remember though, is that there are some elements that do not require the use of closing tag, which are called void tags(eg. < br >, < img >)

Inline vs block tags

Just as the name implies, inline tags keep all the text in a single line, while block tags create kind of an invisible box around the text.

< br >, < b >, < em >, < span > and < img > tags are all examples of inline tags, while < p > and < div > tags are both block ones.
